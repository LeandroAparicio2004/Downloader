# 🎵 Downloader - Python App
Aplicación de escritorio diseñada para facilitar la descarga de contenido multimedia mediante una interfaz gráfica sencilla. Tiene la capacidad de descargar archivos MP3 mediante el buscador o Ingresando el Link (es más preciso si ingresas el link).

### 🛠️ Tecnologías utilizadas
![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![Git](https://img.shields.io/badge/git-%23F05033.svg?style=for-the-badge&logo=git&logoColor=white)

### ✨ Características
- **Interfaz amigable:** Gestión de descargas sin necesidad de usar la consola.
- **Eficiencia:** Implementación de lógica optimizada para el manejo de archivos.
- **Versatilidad:** Soporte para diferentes formatos de audio y video.
